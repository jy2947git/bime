To start:
Install latest Eclipse (with JEE support)
create workspace for example /Users/xxxx/workspace
cd /Users/xxxx/workspace
git clone ssh://git@ssh.github.com:443/jy2947git/bime.git
start Eclipse and import the project "bime" from /Users/xxxx/workspace

You'd need to fix the build path first, to make sure the JDK setup correctly in buildpath.

Eclipse Git plug-in at http://download.eclipse.org/egit/updates-0.8
